## Как добавить ADR

Когда находишь решение проблемы:

1. Запиши контекст: Что было? Почему не работало?
2. Запиши решение: Что сделал и почему это правильно
3. Укажи дату

---

## Пример

```markdown
## ADR-005: React vs vanilla fetch

**Контекст:**
Нужно получить данные с API. Хотел использовать vanilla fetch + useEffect.

**Решение:**
Использовал React Query — автоматический кеш, retry, инвалидация.

**Дата:** 2026-04
```
