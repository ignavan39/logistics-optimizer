## Strict Mode Activation

Если пользователь пишет:
- «сделай фичу Х»
- «реализуй интеграцию с Y»
- «оптимизируй сервис Z»

→ Агент **обязан** сначала:
1. Задать уточняющие вопросы (если не хватает контекста)
2. Предложить план с декомпозицией (по шаблону выше)
3. Дождаться подтверждения плана
4. Только потом писать код
